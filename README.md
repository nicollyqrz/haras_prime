# Haras Prime

## Descrição

Projeto desenvolvido em PHP para simular um sistema interno de um haras.

O sistema possui:

- Busca de cavalos utilizando o método GET;
- Simulação de financiamento utilizando o método POST;
- Cálculo de juros conforme a quantidade de parcelas;
- Verificação de elegibilidade para Cupom VIP.

## Estrutura do projeto

```
├── README.md

```
